# 🔐 Random Password Generator

> A lightweight vanilla JavaScript app that generates random passwords of a custom length, displayed in a clean multi-column layout.

---

## 📌 Description

A simple browser-based tool built with HTML, CSS, and JavaScript. The user inputs a desired password length and clicks **Generate** to produce a random password, which is added to a scrollable list split across multiple columns.

---

## 🚀 Features

- Custom password length (6–30 characters)
- One-click generation
- Multi-column password history list
- Input validation with error feedback
- Google Fonts (Overpass) styling

## ⚙️ Usage

Just open `index.html` in any browser — no installation needed.

```
1. Enter a number between 6 and 30
2. Click "Generate a random password"
3. The password appears in the list below
```
